# AI Prompt Generator Frontend

This is the vanilla HTML/CSS/JS frontend for the AI Prompt Generator. It connects directly to the Flask backend running on `localhost:5000`.

## Tech Stack
- **HTML5**: Single `index.html` structure
- **CSS3**: Vanilla CSS with modern custom properties, dark theme, and grid/flexbox
- **JavaScript**: Vanilla `app.js` (No frameworks, React, or build steps)

## Features
- **Prompt Generator:** Dynamic dropdowns for subject, style, lighting, mood, aspect ratio, and tool.
- **Prompt Library:** Filterable grid of 100+ prompts paginated from the backend API.
- **Copy to Clipboard:** 1-click copy with flash confirmation.
- **Local Fallback:** If the backend is down, it displays hardcoded sample prompts.
- **AdSense Ready:** Clean div blocks setup for easy AdSense integration.

## How to Run

Because this is a vanilla frontend, you can serve it in several ways:

1. **Option A: Static Server**
   Run the backend on port 5000 first:
   ```bash
   cd ..
   node src/server.js
   ```
   Then open `index.html` in your browser. Note: Because of CORS, it's best to run a local dev server for the frontend, for example:
   ```bash
   npx serve .
   ```

2. **Option B: Flask Static Folder Integration**
   Copy `index.html`, `style.css`, and `app.js` to your backend's static/public folder to serve them directly from the Express/Flask app port.

## Google AdSense Setup

To integrate Google AdSense:

1. Add your AdSense `<script>` tag inside the `<head>` of `index.html`:
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXX" crossorigin="anonymous"></script>
   ```

2. Locate the following elements in `index.html`:
   - `<div id="ad-top-banner">...</div>`
   - `<aside id="ad-sidebar">...</aside>`

3. Inside these tags, remove the placeholder `<span>` and insert the `<ins>` tag generated by your Google AdSense dashboard.
   ```html
   <ins class="adsbygoogle"
        style="display:block"
        data-ad-client="ca-pub-XXXXXXXXXXXX"
        data-ad-slot="XXXXXXX"
        data-ad-format="auto"
        data-full-width-responsive="true"></ins>
   <script>
        (adsbygoogle = window.adsbygoogle || []).push({});
   </script>
   ```
